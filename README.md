# BudgetKeeper
> Web-service for keeping budget.

### Сущности базы данных:
* Пользователь (**User**)
* Счет (**Account**)
* Транзакция (**Transaction**)
* Категория(**Category**)
* Шаблон(**Template**)
---
#### СЧЕТ 
##### Типы:
* банковская карта(карты)
* накопительный счет
* наличные
#### Действия над счетами:
* добавить доход\расход
* трансфер между считами
* редактирование
* удаление

#### Транзакция
* тип: доход\расход
* сумма
* источник
* категория
* дата

#### Категория
* название категории

### Шаблоны
* название категории
* сумма расхода
